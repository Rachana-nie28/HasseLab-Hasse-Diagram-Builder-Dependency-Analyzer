## Project Title

**HasseLab – Hasse Diagram Builder and Dependency Analyzer**

---

## Project Description

HasseLab is an interactive web-based application developed for constructing, visualizing, and analyzing Hasse diagrams used in Discrete Mathematics. The project enables users to create partially ordered sets (posets) by entering elements and relations, and automatically generates corresponding Hasse diagrams.

The application verifies mathematical properties such as reflexivity, antisymmetry, and transitivity to determine whether a relation forms a valid poset. It also identifies special elements such as minimal, maximal, least, and greatest elements and checks whether the structure forms a lattice.

Additional features implemented in the project include:

- Manual element and relation input
- Matrix-based relation input
- Preset examples
- Automatic Hasse diagram generation
- Property verification of posets
- Visualization of special elements
- Quiz module for learning
- Learn section with explanations
- Real-world examples and applications
- Interactive node visualization and dependency analysis
The project was developed using HTML, CSS, and JavaScript.
